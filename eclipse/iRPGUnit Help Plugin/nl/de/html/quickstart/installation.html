Installation

Installation der RPGUnit Bibliothek

Zur Installation von RPGUnit wird zunächst die Sicherungsdatei RPGUnit_* auf den Server übertragen und anschließend die Bibliothek RPGUNIT wiederhergestellt. Den Abschluß den Installation bildet das Programm A_INSTALL, welches die RPGUnit Objekte auf der Maschine erstellt.

Übertragen der Sicherungsdatei

Vor dem Übertragen der Sicherungsdatei erstellen Sie bitte eine leere Sicherungsdatei auf ihrem Server: 
Erstellen Sicherungsdatei:
  CRTSAVF FILE(QGPL/RPGUNIT)
Anschließend läßt sich die Sicherungsdatei schnell und einfach per FTP übertragen: 
Öffnen einer DOS Box:
  XP, Win7: Start -> Ausführen -> cmd.exe

Starten einer FTP Sitzung zu ihrem Server:
  FTP your.server.com

Übertragen der Sicherungsdatei:
  binary
  put RPGUnit_v* QGPL/RPGUNIT
  
Beenden der FTP Sitzung:
  quite
Melden Sie sich nun an ihrem Server an und stellen Sie die Bibliothek RPGUNIT wieder her und fügen Sie die Bibliothek ihrer Bibliotheksliste hinzu: 
  RSTLIB SAVLIB(RPGUNIT) DEV(*SAVF) SAVF(QGPL/RPGUNIT)
  ADDLIBLE RPGUNIT
Starten Sie jetzt den PDM für die Datei RPGUNIT1 der Blibliothek RPGUNIT und erstellen Sie die RPGUNIT Objekte: 
  CRTBNDCL PGM(RPGUNIT/A_INSTALL) SRCFILE(RPGUNIT/RPGUNIT1) SRCMBR(*PGM)
  CALL PGM(RPGUNIT/A_INSTALL) PARM('RPGUNIT')
Damit ist die Installation abgeschlossen.

Installieren des Plugins

Verwenden der Update Site

Die URL der Update Site ist:

http://www.tools400.de/rpgunit/update/rdp8.0/

Verwenden des 'dropins' Verzeichnis

Gehen Sie zunächst zum 'dropins' Verzeichnis ihrer Rational Developer for Power (RDP) Installation. Normalerweise ist dies:

c:\Program Files\IBM\SDP\dropins\

Erstellen Sie nun eine Text Datei 'de.tools400.rdp.rpgunit.link' mit folgendem Inhalt (Beispiel) in ihrem 'dropins' Verzeichnis: 
path=C:/Users/Joe/Documents/RPGUnit Plugin/local_updatesite/
Abschließend kopieren Sie das Verzeichnis 'local_updatesite' der zip Datei in das Verzeichnis, welches Sie soeben bei 'path' angegeben haben. Im Ergebnis sollte die Verzeichnisstruktur wie folgt aussehen: 
C:/Users/Joe/Documents/
 +-- RPGUnit Plugin
      +-- local_updatesite
           +-- plugins
                +-- de.tools400.rpgunit.core.help_*.jar
                    de.tools400.rpgunit.core_*.jar
                    de.tools400.rpgunit.isphere_*.jar
                    de.tools400.rpgunit.spooledfileviewer_*.jar
           +-- features
                +-- de.tools400.rpgunit.core.help_*.jar
                    de.tools400.rpgunit.core_*.jar
                    de.tools400.rpgunit.isphere_*.jar
                    de.tools400.rpgunit.spooledfileviewer_*.jar
Bitte achten Sie darauf bei der Pfadangabe einer 'slash' (/) und keinen 'back slash' (\) zu verwenden und vergessen Sie auch nicht den abschließenden 'slash' hinter 'local_updatesite'.

Damit ist die Installation abgeschlossen. Sollte RDP aktiv sein, müssen Sie RDP zur Aktivierung des Plugins neu starten.

Bitte vergessen Sie nicht die Bibliothek RPGUNIT in die Bibliotheksliste ihrer Remote Connection aufzunehmen.
