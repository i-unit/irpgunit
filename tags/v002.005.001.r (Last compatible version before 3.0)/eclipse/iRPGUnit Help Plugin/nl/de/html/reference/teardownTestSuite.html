tearDownSuite (tear down test suite)

Syntax:

tearDownSuite();

Beschreibung:

Beendet die Test Suite und wird einmalig zu Beginn des Testlaufs aufgerufen.

In dieser Prozedur können abschließende Aufräumarbeiten durchgeführt werden und zum Beispiel die Ressourcen, die innerhalb von setUpSuite() zugeordnet wurden wieder freigegeben werden.
