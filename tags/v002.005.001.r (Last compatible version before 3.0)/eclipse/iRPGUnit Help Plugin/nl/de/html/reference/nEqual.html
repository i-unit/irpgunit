nEqual (boolean equal)

Syntax:

nEqual( erwarteterWert : tatsächlicherWert [: 'fieldName');

Beschreibung:

Vergleicht die beiden boolschen Werte oder Ausdrücke 'erwarteterWert' und 'tatsächlicherWert' miteinander. Der Testfall gilt als fehlgeschlagen, wenn die beiden Werte unterschiedlich sind.

Beispiel: 

nEqual( *off : 1=2 );

// No exception message => OK.

nEqual( *on : 'Hello'='HelloWorld' );

// Never reached!
