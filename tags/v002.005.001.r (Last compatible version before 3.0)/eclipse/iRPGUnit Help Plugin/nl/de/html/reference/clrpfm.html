CLRPFM (clear physical file member)

Syntax:

clrpfm( 'datei' [: 'teildatei' );

Beschreibung:

Führt den Befehl CLRPFM für die in teildatei angegebenen Teildatei der in datei angegebenen Datei ohne weitere Parameter aus.

Achtung: Die zu löschende Datei muss sich in der aktuellen Bibliothek (*CURLIB) befinden!

Beispiel: 

// Clears member 'TEST1' of file 'TESTWERTE'.

clrpfm( 'TESTVALUES' : 'TEST1' );
