clearAssertCounter (clear assertions counter)

Syntax:

clearAssertCounter();

Beschreibung:

Setzt die Anzahl der bisher durchgeführten Zusicherungen der aktuell laufenden Testprozedur zurück auf 0.

Beispiel: 

// Setup.

assertCallCnt = getAssertCalled();

// Execution.

assert( *on: *blank );
iEqual( 4: 2+2 );
aEqual( 'Hello': 'Hello' );

// Controls.

iEqual( assertCallCnt + 3 : getAssertCalled() );
clearAssertCounter();
iEqual( 0 : getAssertCalled() );
