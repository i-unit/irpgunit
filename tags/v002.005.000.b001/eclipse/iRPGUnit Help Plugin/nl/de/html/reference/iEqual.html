iEqual (integer equal)

Syntax:

iEqual( erwarteterWert : tatsächlicherWert [: 'fieldName');

Beschreibung:

Vergleicht die beiden Integer Werte erwarteterWert und tatsächlicherWert miteinander. Der Testfall gilt als fehlgeschlagen, wenn die beiden Werte unterschiedlich sind.

Beispiel: 

iEqual( 123 : 123 );

// No exception message => OK.

iEqual( 123 : 456 );

// Never reached!
