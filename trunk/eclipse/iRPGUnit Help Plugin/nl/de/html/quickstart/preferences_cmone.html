Voreinstellungen CMOne Integration

Diese Seite enthält Optionen zur Integration von iRPGUnit in CMOne, der ChangeManagement Software der Task Force IT-Consulting GmbH, Deutschland.

Diese Voreinstellungen sind nur verfügbar, wenn das iRPGUnit CMOne Integration Feature installiert ist.
